		Блок

Блок

независимый компонент, который может использоваться поторно

.name.block-name

Элемент

составная часть, вложенная в блок,
без блока-родителя существовать не может

.block-name__element.block-name__element-name

Название описывает "что это", а не "какой или как выглядит?"

Элемент. 3 основных правила

  1. Вложенность
  2. Принадлежность
  3. Необязательность

Модификатор

отвечает, описывает внешний вид, состояние или поведение.

		.block-name__element--mode //булевый
		.block-name__element--key-value //ключ-значение
		.block-name--mode //модификатор блока
		.text--color-red //bad
		.text--alert //good

Основные правила:

  1. Не исопользуем id
  2. Блоки делаем независимыми от контекста
  3. Не обращаемся в css по тегу*
  4. Не используем каскад*
  5. Не используем глобальные стили*

Поговорим о коллегах

Поговорим о коллегах — контент-менеджер

  • Он не должен знать о БЭME!
		.block {
			a {/* какие-то стили */}
			p {/* какие-то стили */}
			ul {/* какие-то стили */}
		}

Поговорим о коллегах — бекенд-девелопер

  • Используй понятный неймспейс для классов
  • Следи за структурой многостраничного сайта, не переусложняй
  • Уникальные классыдля каждой страницы — одумайся!
  • Дроби блоки и переиспользуй чаще
  • не верстай "щоб було"©

Резюмируем ошибки:

  • Элемент элемента
  • Элемент вне блока
  • Контекстная зависимость блоков
  • Контекстная зависимость элемента
  • Наглый модификатор элемента (кроме самого себя трогает еще и другие элементы этого же блока)

Резюмируем ошибки:

  • Жирные блоки
  • Блок, вложенный в себя
  • Использование стилевых классов в JS
  • Использование классов-хелперов «как-бутстрап»: <ul class="list pointer left colorred">
  • Неиспользование классов-хелперов (модификаторы --float_left)
  • Одноразовые элементы (создание одинаковых элементов с разными именами типа .list__item1,.list__item2,…)

Резюмируем ошибки:

  • Oldschool reborn (высокая специфичность и класс только на блоке)
  • Looooooooong naming
  • Модификация элементами (смена имени элемента вместо добавления модификатора)
