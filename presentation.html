		Блок
		Элемент

Блок

независимый компонент, который может использоваться поторно

.name.block-name

Элемент

составная часть, вложенная в блок,
без блока-родителя существовать не может

.block-name__element.block-name__element-name

Название описывает "что это", а не "какой или как выглядит?"

Элемент. 3 основных правила

  1. Вложенность
  2. Принадлежность
  3. Необязательность

Модификатор

отвечает, описывает внешний вид, состояние или поведение.

		.block-name__element--mode //булевый
		.block-name__element--key-value //ключ-значение
		.block-name--mode //модификатор блока
		.text--color-red //bad
		.text--alert //good

Основные правила:

  1. Не исопользуем id
  2. Блоки делаем независимыми от контекста
  3. Не обращаемся в css по тегу*
  4. Не используем каскад*
  5. Не используем глобальные стили*

Схемы именования

		block__element--mode //two dashes
		block__element_mode //standart
		MyBlock__SomeElement_modeName//CamelCase
		blockName-elment--modeName //Стиль без подчеркиваний
		_available //no-namespace
